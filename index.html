How To Create 3D Music Video

Click Choose file to upload music

Select Visualizer Icon

Click Play to start music

Use keyboards to Fly Controls navigation

WASD move, R up, F down, QE roll, up/down pitch, left/right yaw

Or select programmed camera movement. Try Picture in Picture camera

Clink Open Controls to control object settings

Watch enjoy 3D Music Visualization

Yes you can make your own music video! Make one of kind your own music video. Add images and control visualizer. ..

Clink Start Record when it is ready

Clink Stop Record when it is done. Recorded mv file will be downloaded

Recorded webm file will be downloaded
